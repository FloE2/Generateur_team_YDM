📖 Logique de composition
3 modes de composition disponibles :

⚖️ Équilibrées — Distribution en serpentin : chaque équipe a un mélange de tous les niveaux. Idéal pour des matchs équilibrés.

★ Similaires — Les élèves du même niveau sont regroupés ensemble (les meilleurs avec les meilleurs, les débutants avec les débutants). Idéal pour un travail différencié.

🎲 Aléatoire — Répartition aléatoire sans critère de niveau.

▶ En mode Séparé, l'appli calcule automatiquement toutes les configurations valides et vous laisse choisir. Tous les élèves présents sont toujours répartis (tailles ±1 joueur).
